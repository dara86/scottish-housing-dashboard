This personal project was to create an interactive dashboard designed to analyse and visualise Scottish property price trends **from 2005 to 2025**. 

Focuses on regional comparisons, property type gaps (Flat/Semi-Detached/Fully Detached), and historical performance across all Scottish local authorities.

# Live Demo
To access the live dashboard, [click here!](https://scottish-housing-dashboard.streamlit.app)


# Data Source
The data used in this dashboard is sourced from the [UK Government House Price Index (HPI)](https://www.gov.uk/government/statistical-data-sets/uk-house-price-index-data-downloads-december-2025).
